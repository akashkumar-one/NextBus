# Find Next Bus

The application will output the next bus departure time for a given route and stop.

### Prerequisites

```
memcached
```

### Installing

```
npm install
node index.js arg1 arg2 arg3
```
### Example
```
node index.js "Brklyn Center - Fremont - 26th" "Brooklyn Center Transit" NORTHBOUND
```
